# 1. Nr. / Buerger
Nr. 8 - Buerger 8

# 2. Ursprung Stellungnahme (Kurzfassung)
Gefaehrdung von Wildwechseln und Vogelarten, Verlust von Rueckzugsraeumen, Forderung nach naturvertraeglichen Alternativstandorten.

# 3. Im Entwurf beruecksichtigt (mit Evidenz/Fundstellen)
- Erhebung von Wildwechseln ist Bestandteil des Artenschutzfachbeitrags (Anlage 3, Zeilen 248-249 sowie 590-593).
- Zaundurchlaesse/Wildkorridor werden verbindlich vorgesehen (Planteil C, Zeilen 764-771; Planteil AB, Zeilen 231-234).
- Artenbezogene Vermeidungs-/CEF-Massnahmen inkl. Monitoring fuer Feldlerche/Wiesenpieper/Kranich/Zauneidechse sind festgelegt (Anlage 1, Zeilen 67-69, 81-86, 186-187, 383, 473, 631, 966, 1058).
- Standortalternativen sind im Entwurf geprueft (Planteil C, Zeilen 310-315).

# 4. Bewertung
Teilweise berücksichtigt

# 5. Offener Punkt
Ob die vorgesehenen Korridore die lokale Funktion als Hauptwildwechsel vollumfaenglich erhalten, bleibt prognostisch.

# 6. Empfehlung
Vorhabensbegleitendes Monitoring mit klaren Nachsteuerungs-Triggern fuer Wildkorridor-Funktion vertraglich schaerfen.

# 7. Unsicherheiten
Wirksamkeit von Wildkorridoren ist quellenimmanent prognosebehaftet; kein vollstaendiger Vorher-Nachher-Nachweis im Entwurf.
