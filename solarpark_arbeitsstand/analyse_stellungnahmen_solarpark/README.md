# Analyse Stellungnahmen Solarpark

Zweck: Nachvollziehbare Bewertung, wie gut die Bürger-Stellungnahmen in den neuen Planunterlagen (`pu/`) berücksichtigt sind.

## Struktur

- `detail/` – Detaillierte Auswertung je Bürger (eine Datei pro Bürger)
- `tabellen/` – Gesamtübersicht als Markdown + CSV
- `cache/` – Persistente Zwischenstände (Extrakte, Zuordnungen, Evidenz)
